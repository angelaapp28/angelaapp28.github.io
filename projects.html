3D Point Cloud Registration

Aligning 3D medical scans using iterative algorithms

Project Overview

When doctors need to align 3D medical scans (like CT or MRI images) taken at different times or from different angles, they face a challenging computational problem. This project solves that problem by automatically finding the best alignment between different 3D scans.
